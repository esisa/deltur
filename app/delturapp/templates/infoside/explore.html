deltur.no gir deg uante muligheter til å dele dine turer

Del turen din enkelt på deltur.no

Last opp din GPS-fil og del den raskt og enkelt på deltur.no. Du kan også velge avansert deling og dele punkter og bilder i tillegg til din GPS-fil.

Du kan også "legge sammen" ulike turer med enkelt å bruke plusstegnet i URL'en slik som denne turen deltur.no/3+4+5

Del turen på sosiale medier

Med deltur.no kan du vise alle dine venner hvor du har vært i dag!

Nå trenger du ikke lenger forklare hvor du har vært på tur, hvor mange bakker du har løpt eller hvor du startet turen. All informasjon ligger i kartet og du kan dele det med dine venner på Facebook, Twitter, Google+ eller på e-post.

Alt kun et klikk unna.

Vis turen på egne nettsider

Med deltur.no kan du dele dine turer enkelt på dine egne nettsider.

Alle turer på deltur.no kan deles på din egen nettside ved å kopiere en enkel kode som du limer inn på ditt eget nettsted. Det betyr at du svært enkelt kan tilby kart på alle dine turforslag eller vise turen du har gått på bloggen din.

Velg sjøl størrelsen på kartet, resten skjer automatisk!

Bruk deltur.no API som lagringsplass for dine turer

Det er enkelt å implementere deltur.no i egne systemer gjennom bruk av deltur.no API.

deltur.no tilbyr et API der du kan lagre dine turer og vise de fram slik du ønsker det. Gjennom API'et kan du også laste ned turen i mange ulike formater.

Bruk deltur.no API gjennom ditt eget nettsystem og få et fullverdig lagringssystem for dine turer nesten gratis. Enkelt er det også!

Style om dine turer for å passe din profil

Velg en kartløsning som ikke bryter med ditt eksisterende design.

Hvorfor skal du legge til et kart på ditt eget nettsted som bryter fullstendig med din visuelle profil? Med deltur.no kan du style alt til å se akkurat slik du ønsker deg. Du velger sjøl hva slags type markører du vil bruke og farge på linjene i kartet. Ikke bare det, du kan også legge inn et eget bakgrunnskart som er spesialtilpasset ditt design.

Resultatet er et kart som passer 100% med din visuelle profil!
