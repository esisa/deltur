Del og miks turer

Med deltur.no kan du mikse linjer og steder som du selv vil. Du kan bygge opp turen med linjer og steder slik du selv ønsker det.

Lag ditt eget design

Med deltur.no kan du vise dine turer med ditt eget design. Linjer, punkter og bakgrunnskartet kan ha farger tilpasset ditt bruk.

Mer glad i API enn i GUI

Med deltur.no kan du laste opp turer gjennom grensesnittet på deltur.no. Men du kan også integrere deltur.no i dine egne systemer ved å bruke deltur.no API.


Hva er deltur.no?

deltur.no gir deg en enkel måte å dele dine turer på. Du trenger ikke lengre å forklarer hvor du har vært på tur. Nå kan du legge inn turen direkte på deltur.no og dele med dine venner.

Har du en hjemmeside eller blogg der du vil vise fram din favorittur så legg turen inn på deltur.no og velg å enkelt dele denne turen på din egen side. Du kan styre utseendet på kart og tur slik at det passer perfekt inn på din egen side.

Er du mer teknisk anlagt kan du bruke deltur.no API for å integrere deltur.no inn i dine egne produkter.

Hva kan du gjøre med deltur.no?

Del turen din enkelt på deltur.no

Last opp din GPS-fil og del den raskt og enkelt på deltur.no. Du kan også velge avansert deling og dele punkter og bilder i tillegg til din GPS-fil.

Du kan også "legge sammen" ulike turer med enkelt å bruke plusstegnet i URL'en slik som denne turen beta.deltur.no/101+102+103



Del turen på sosiale medier

Med deltur.no kan du vise alle dine venner hvor du har vært i dag!

Nå trenger du ikke lenger forklare hvor du har vært på tur, hvor mange bakker du har løpt eller hvor du startet turen. All informasjon ligger i kartet og du kan dele det med dine venner på Facebook, Twitter, Google+ eller på e-post.

Alt kun et klikk unna.



Bruk deltur.no til å vise fram turen på dine sider





Vis turen på egne nettsider

Alle registrerte turer på deltur.no kan deles på din egen nettside ved å kopiere en enkel kode som du limer inn på ditt eget nettsted. Det betyr at du svært enkelt kan tilby kart på alle dine turforslag eller vise turen du har gått på bloggen din. Velg selv størrelsen på kartet, resten skjer automatisk!


Bruk deltur.no API som lagringsplass for dine turer

deltur.no tilbyr et API der du kan lagre dine turer og vise de fram slik du ønsker det. Gjennom API'et kan du også laste ned turen i mange ulike formater. Bruk deltur.no API gjennom ditt eget nettsystem og få et fullverdig lagringssystem for dine turer nesten gratis. Enkelt er det også!


Style om dine turer for å passe din profil

Velg en kartløsning som ikke bryter med ditt eksisterende design. Hvorfor skal du legge til et kart på ditt eget nettsted som bryter fullstendig med din visuelle profil? Med deltur.no kan du style alt til å se ut akkurat slik du ønsker deg. Du velger selv hva slags type markører du vil bruke og farge på linjene i kartet. Ikke bare det, du kan også legge inn et eget bakgrunnskart som er spesialtilpasset ditt design. Resultatet er et kart som passer 100% med din visuelle profil!
